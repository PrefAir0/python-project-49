## Описание проекта

Данный проект представляет собой: консольное приложение с набором математических игр brain games
В brain games входит:
    ``brain_prime``         - Определи простое ли число
    ``brain_gcd``           - Найди наибольший общий делитель
    ``brain_progression``   - Вычисли пропущенное число в прогрессии
    ``brain_even``          - Определи является ли число четным
    ``brain_calc``          - Выполни простые орифметические операции


## Технологии и инструменты

* **Язык программирования:** Python 3.13+
* **Менеджер пакетов и зависимостей:** uv
* **Качество кода проверенно с помощью:** ruff, SonarCloud 

## Инструкция по установке

* Установите пакетный менеджер uv
* Установите зависимости командой ``uv sync``
* Запустите выбранную игру ``uv run [название_игры]``
